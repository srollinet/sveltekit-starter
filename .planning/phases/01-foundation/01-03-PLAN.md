# Svelte 5 Runes — key patterns for this plan:

# - Reactive state: let theme = $state('light')

# - Derived: let isDark = $derived(theme === 'dark')

# - Side effects: $effect(() => { localStorage.setItem('theme', theme) })

# - NO: writable() stores, $: reactive statements (those are Svelte 4)

**Imports and setup:**

- Import `../app.css` to ensure Tailwind + DaisyUI styles load on every page
- Import `{ onMount }` from `'svelte'`
- Use Svelte 5 `$state()` rune for the theme value (NOT writable() store)
- Use `$effect()` rune to persist theme to localStorage and apply to DOM

**Theme logic (implements D-07, D-08):**

```svelte
<script lang="ts">
  import '../app.css';
  import { onMount } from 'svelte';

  let theme = $state('light');

  onMount(() => {
    // D-08: Check localStorage on initial load; default to 'light' if absent
    const saved = localStorage.getItem('theme');
    if (saved === 'light' || saved === 'dark') {
      theme = saved;
    }
    document.documentElement.setAttribute('data-theme', theme);
  });

  $effect(() => {
    // D-07: Persist and apply theme on every change
    localStorage.setItem('theme', theme);
    document.documentElement.setAttribute('data-theme', theme);
  });

  function toggleTheme() {
    theme = theme === 'light' ? 'dark' : 'light';
  }
</script>
```

## STRIDE Threat Register
