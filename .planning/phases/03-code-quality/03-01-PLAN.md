## Tasks

### Task 1 — Install prettier-plugin-tailwindcss and update .prettierrc

**What:** Add Tailwind class sorting to Prettier. Must load after `prettier-plugin-svelte`.

```bash
pnpm add -D prettier-plugin-tailwindcss
```

Update `.prettierrc` to add the plugin last:

```json
{
  "trailingComma": "all",
  "singleAttributePerLine": true,
  "plugins": ["prettier-plugin-svelte", "prettier-plugin-tailwindcss"],
  "overrides": [
    {
      "files": "*.svelte",
      "options": {
        "parser": "svelte"
      }
    }
  ]
}
```

Then re-format the entire codebase so all files reflect the new plugin:

```bash
pnpm run format
```

**Verify:** `pnpm run lint` still passes after reformatting.

---

### Task 2 — Simplify the lint script to ESLint only

**What:** Remove the `prettier --check .` prefix from the `lint` script.

In `package.json`, change:

```json
"lint": "prettier --check . && eslint ."
```

to:

```json
"lint": "eslint ."
```

**Rationale (from CONTEXT.md):** `eslint-config-prettier` already disables conflicting ESLint rules. Prettier is enforced on commit via lint-staged. Keeping `prettier --check` in lint is redundant and slows CI.

The `format` script and lint-staged prettier step are unchanged.

---

### Task 3 — Install Knip and configure it

**What:** Add dead code detection configured for SvelteKit's file-based routing.

```bash
pnpm add -D knip
```

Add `knip.config.ts` at project root:

```typescript
import type { KnipConfig } from 'knip';

const config: KnipConfig = {
  entry: ['svelte.config.js', 'vite.config.ts', 'playwright.config.ts', 'src/hooks.server.ts'],
  project: ['src/**/*.{ts,svelte}', '*.{js,ts}'],
};

export default config;
```

> **Note:** The Knip Svelte plugin auto-detects SvelteKit file-based routing (`src/routes/**`) and `$lib` exports. The `entry` array above adds top-level config files the plugin may miss. Use default (non-strict) mode — the starter ships intentional example code that would generate noise in strict mode.

Add `knip` script to `package.json`:

```json
"knip": "knip"
```

If `pnpm run lint` reports errors after the `prettier --write` pass, fix them before committing.
