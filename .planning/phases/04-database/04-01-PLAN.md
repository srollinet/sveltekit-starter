**1. Create `src/lib/server/db/schema/posts.ts`** (per D-03, D-04, D-05):

```typescript
import { pgEnum, pgTable, text, timestamp, uuid } from 'drizzle-orm/pg-core';
import { uuidv7 } from 'uuidv7';

export const postStatusEnum = pgEnum('post_status', ['draft', 'published', 'archived']);

export const posts = pgTable('posts', {
  id: uuid('id')
    .primaryKey()
    .$defaultFn(() => uuidv7()),
  title: text('title').notNull(),
  body: text('body'),
  status: postStatusEnum('status').notNull().default('draft'),
  createdAt: timestamp('created_at', { withTimezone: true }).notNull().defaultNow(),
  updatedAt: timestamp('updated_at', { withTimezone: true })
    .notNull()
    .defaultNow()
    .$onUpdateFn(() => new Date()),
});
```

**2. Create `src/lib/server/db/schema/index.ts`** (barrel file):

```typescript
export * from './posts.js';
```

Note: Use `.js` extension for ESM compatibility with SvelteKit's TypeScript resolution.

**3. Create/update `src/lib/server/db/index.ts`** (per D-07, D-08):

```typescript
import { drizzle } from 'drizzle-orm/postgres-js';
import postgres from 'postgres';
import { env } from '$lib/server/env';

const client = postgres(env.DATABASE_URL);
export const db = drizzle({ client });
```

CRITICAL per D-08: Import `env` from `$lib/server/env` (the Zod-validated module), NOT from `$env/dynamic/private` directly, and NOT from `process.env`.

**4. Create/update `drizzle.config.ts`** at project root (per D-06, DB-03):

```typescript
import { defineConfig } from 'drizzle-kit';

export default defineConfig({
  out: './drizzle',
  schema: './src/lib/server/db/schema/*',
  dialect: 'postgresql',
  dbCredentials: {
    url: process.env.DATABASE_URL!,
  },
});
```

Note: drizzle.config.ts uses `process.env.DATABASE_URL!` (not `$lib/server/env`) because drizzle-kit runs as a plain Node.js CLI outside SvelteKit context — `$lib` aliases are not available.

**5. Delete scaffold artifacts** that were replaced:
